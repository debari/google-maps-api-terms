          Frequently-asked questions
        
              What will happen to my Google Maps Engine data?
            
              All data stored with Google Maps Engine will be systematically deleted from Google servers.
            
              Will there be any way to retrieve my subscription data from Google in the future?
            
              No -- all data stored in Google Maps Engine will be permanently and irrevocably deleted. Google will
              not be able to recover any Google Maps Engine data for any user after February 1st, 2016.
            
              What are other options for hosting data?
            
              Here are some possible options to consider hosting data with the Google Maps Premium Plan and Google Cloud Platform.
            
Building a store locator with the Google Maps API and Cloud SQL
Building spatial applications with Google Cloud SQL and Google Maps API
Serving raster layers on Google Cloud Platform
